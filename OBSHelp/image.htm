Image


Browse

...lets you select the desired Image. Animated GIF files and files with transparency are supported (png for example)

Color

Allows you to tint the image. White is completely normal.

Opacity

Change the visibility of your Image.

Check for file changes

Activate this option to allow any file changes that occur to be applied in real time.

Chroma Key

Use Color Key

Activates the chroma keying.

Color

Select the desired Color you want to chroma key out of your Image. (Make transparent)

Similarity

How much the selected color is allowed to vary. A higher value, means it can vary more.

Blend

Specified how much to smooth out the edges chroma keyed color. A higher value means smoother edges.
