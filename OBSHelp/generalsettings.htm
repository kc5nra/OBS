General


In the general settings you can switch the Language and also add, rename or remove Profiles.

To create a new profile, write the name you would like for it and then click Add to create it. It will use the settings of the previously selected profile as its defaults. Change all settings you like and close OBS to save your new profile, or select File -> Save from the main window. For additional profiles, do the same. Make sure to click Add before you beging changing settings. Or your old profile will be overwritten.

You can also activate the Notification Area Icon and the minimize to Notification Area option here.
